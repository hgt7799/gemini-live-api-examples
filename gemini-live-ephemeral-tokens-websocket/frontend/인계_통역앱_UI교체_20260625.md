## 무엇을 했는가

일반 사용자용 **라이트 'A 소프트 카드' 통역 화면**으로 교체했다.

- 흰 배경 + 모바일 앱 셸 (max-width 480px)
- 왼쪽 파란 'English' 카드 = 통역된 영어
- 오른쪽 회색 '한국어' 카드 = 내가 말한 한국어
- 상단 ⚙ 기어 → 고급 설정 서랍 (CSS-only 토글)
- 하단: [연결] / [종료] 버튼 + 파란 원형 마이크 버튼

## 확인된 제약 (수정 불가)

- **자막 두 토막 갈라짐** → 해결됨 (커밋 `c9dd5f7`)
- **통역 모델이 짧은 말을 자연스러운 영어 문장으로 늘리거나, 안 들은 내용을 채우는 현상** → 수정 불가로 결론.
  - (1) `gemini-3.5-live-translate-preview`는 전용 번역 모델이라 시스템 프롬프트를 안 받음.
  - (2) `translation_config`는 `targetLanguageCode` / `echoTargetLanguage` 두 옵션뿐이라 추측 제어 옵션이 없음.
  - (3) 연속 스트림 처리 방식이라 발화 끊김을 무시하고 동시통역사처럼 문장을 다듬는 게 모델 설계임 (2026-06-25 공식 문서로 확인).
  - 향후 직역이 꼭 필요하면 turn 기반 모델(`gemini-3.1-flash-live-preview`)로 교체해야 하나, 번역 품질·구조가 전부 달라지는 대공사 → 보류.
